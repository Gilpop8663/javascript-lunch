## 📌 실행 방법

1. git clone을 한다.

2. repository 로 폴더 이동

```
cd javascript-lunch
```

3. step2의 브런치로 이동

```
git checkout step2
```

4. dependency 다운로드

```
npm install
```

5. 프로젝트 시작

```
npm run start
```
